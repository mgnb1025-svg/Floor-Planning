# TileArchitect AI Pro 🛠️📐

**TileArchitect AI Pro** is a high-fidelity flooring companion designed to bridge the gap between amateur DIYers and professional contractors. By utilizing computer vision and manual input logic, it ensures perfect layouts and accurate material estimates.

## 🚀 Key Features
* **AI Diagnostic Scan:** Real-time visual analysis to detect wall variances and out-of-square corners.
* **Smart Waste Optimizer:** Dynamic material calculation based on room complexity scores (5% to 20% waste logic).
* **Contractor Dashboard:** A dedicated Pro tier for branded PDF estimates and project management.
* **Hybrid Monetization:** Built-in logic for Pay-Per-Job unlocks ($0.10/sqft) and Monthly Subscriptions ($25/mo).

## 🛠️ Technical Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+).
* **Graphics:** HTML5 Canvas API for real-time grid rendering.
* **Deployment:** Designed to be wrapped with Capacitor for iOS/Android distribution.

## 📐 Recommended Hardware for Testing
For maximum precision and to calibrate the AI logic, this software is optimized for use with:
* **Leica Disto D2:** Industry-standard laser measurement for manual data verification.
* **iPad Pro M4:** Utilizing the M4 LiDAR sensor for 3D room mesh generation.
* **Dremel Ultra-Saw:** Recommended for precision "rescue cuts" identified by the Waste Optimizer.
